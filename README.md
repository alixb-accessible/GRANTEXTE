# Grantexte - Service de partage de liens et contenus

## Description

Grantexte est un outil unique qui combine deux fonctionnalités essentielles :

1. **Partage de contenu texte** : Créez des liens de partage pour vos textes, articles, notes ou tout autre contenu textuel. Cette fonction est très utile lorsque vous avez un communiqué de presse long en jpeg et que vous souhaitez compléter la section ALT TEXT. Si vous connaissez l'outil https://alixb-accessible.github.io/Outil-de-descriptions-d-images-accessibles-/ vous voyez tout à fait ce que cela veut dire.
2. **Réducteur d'URLs** : Raccourcissez vos liens trop longs pour les réseaux sociaux.

## Fonctionnalités principales

- **Hébergement gratuit via GitHub Gists** : Vos contenus sont stockés gratuitement et de manière permanente sur GitHub
- **Liens permanents** : Les liens créés fonctionnent partout dans le monde, sans limitation de temps
- **Pas de compte requis** : Utilisez le service sans inscription
- **Partage social intégré** : Boutons de partage directs vers Twitter, Bluesky et Facebook
- **Redirection automatique** : Pour les liens raccourcis, redirection automatique après 3 secondes
- **Export et copie** : Copiez facilement vos contenus ou les liens générés

## Accessibilité

Grantexte a été conçu avec l'accessibilité au cœur du projet :

- **Navigation au clavier complète** : Tous les éléments sont accessibles via le clavier (Tab, Entrée, Échap)
- **Compatibilité lecteurs d'écran** : Utilisation d'ARIA et de labels descriptifs
- **Contrastes élevés** : Textes et boutons respectent les normes WCAG
- **Pas d'émojis dans l'interface** : Pour éviter les problèmes avec les lecteurs d'écran
- **Focus visible** : Indication claire de l'élément actif
- **Liens d'évitement** : Accès rapide au contenu principal
- **Intégration de la toolbar Faites-Mieux** : Outils d'accessibilité supplémentaires

## Comment ça marche ?

### Mode "Partager du contenu"
1. Donnez un titre à votre contenu (optionnel)
2. Collez ou écrivez votre texte
3. Cliquez sur "Créer le lien de partage"
4. Un lien court est généré automatiquement (ex: `votre-site.com?t=abc123`)
5. Partagez ce lien où vous voulez

### Mode "Réduire un lien"
1. Donnez un titre à votre lien (optionnel)
2. Collez l'URL longue à raccourcir
3. Cliquez sur "Créer le lien raccourci"
4. Un lien court est généré (ex: `votre-site.com?u=xyz789`)
5. Lorsque quelqu'un clique dessus, il est automatiquement redirigé vers l'URL originale

## Technologies utilisées

- **HTML5** / **CSS3** : Interface moderne et responsive
- **JavaScript vanilla** : Pas de framework, code léger et rapide
- **GitHub Gists API** : Stockage gratuit et permanent des contenus
- **Police Lexend** : Typographie optimisée pour la lisibilité
- **Toolbar Faites-Mieux** : Outils d'accessibilité avancés

## Installation

Ouvrez directement dans un navigateur.
Aucune configuration nécessaire.


## Limitations

- **Contenu public** : Tous les contenus créés via GitHub Gists sont publics
- **Texte uniquement** : Pas de support pour les fichiers PDF, images ou autres médias
- **Dépendance à GitHub** : Si GitHub Gists est indisponible, le service ne fonctionne pas
- **Pas de suppression** : Une fois créés, les Gists ne peuvent pas être supprimés via l'interface (fonctionnalité GitHub)
